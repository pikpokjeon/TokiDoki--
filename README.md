### 프로젝트 시작하기

1. 백엔드
```
npm run start
```
2. 프론트엔드
```
npm run serve
```

### 기술스택
#### 백엔드
1. Graphql Yoga 
#### 프론트엔드
1. Graphql Apollo-Client
2. Graphql Apollo-Link
3. Vue-Apollo
4. Vuex
5. 
### 유의사항
- 백엔드 포트(8282) 와 프론트엔드 포트(8080)를 시작 중입니다.
